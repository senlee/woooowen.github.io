####php的浮点运算一直是有问题得.

```js

example1:
$a = floor((0.1+0.7) * 10);
//返回的结果并不是8,而是7
echo $a;

```

---------------------------------------

####so use round()

```js

$a = round((0.1+0.7) * 10);
//返回的结果 = 8
echo $a;

```
```js

example2:
$a = 9 - 5.1;
$b = 3.9;
var_dump($a == $b);
//返回结果是false
```
---------------------------------------
####so use round()

```js

$a = 9 - 5.1;
$b = 3.9;
var_dump(round($a, 2) == round($b, 2));
```
